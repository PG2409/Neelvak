# Release Notes - Neelvak AIOS v1.3
## Overview
Neelvak AIOS v1.3 represents the stable, production-hardened microkernel baseline. It enforces absolute security borders, high-throughput prioritized event loops, and robust error recovery capabilities.

## Key Features
- **Deterministic 10-Pass Compiler**: Translates natural language queries into signed, immutable execution plans.
- **Unified Runtime Contract**: Standardizes A-E runtimes (Competitive, Standard, Micro, Direct, Retrieval) under a shared contract.
- **Hardened Sandbox**: Directory traversal-resistant filesystem tools and sandboxed Python eval checks.
- **Provider-Independent Model Router**: Implements dynamic capability resolution, provider diversity weights, and health anti-flapping mitigations.
- **Resource Recovery**: Dynamically recycles workspace files and purges completed workflow tracking references.
